v1.1.0
- 완전 자동화 오케스트레이터(AutomationOrchestrator) 신규: 게시판→제목→본문→사진→태그를
  [전체 자동 실행 시작] 1회로 순차 자동 진행, 발행 직전 정지는 그대로 유지
- 상태머신에 자동화 세부 20단계 추가(READY_TO_AUTOMATE~VERIFYING_TAGS), 인접 전이만 허용하는
  advanceTo(), 재시도 초과/타임아웃 시 자동 PAUSED
- 선택자 규칙(selector_rules.json) 구조화 재구현: 화면별 후보 조합, 스키마 검증, 정식/이전/기본 3단계 관리
- 사진 자동 선택 파이프라인 신규(ImageConverter/SavedImageManifest/PhotoSelectionPlanner),
  세션별 고유 파일명, 대표→본문 순서, 중복선택/개수불일치 방어
- 게시판 자동 선택을 실제 화면 텍스트 기준으로 확장, 미일치 시 사용자 직접 선택 감지·기록
- 오류 감지 범위 확장(화면꺼짐/앱전환/권한해제) + 세션 영구 저장 + 앱 재실행 후 이어하기 UI
- 플로팅 컨트롤을 일시정지/이어하기/즉시중지 중심으로 재구성, 개별 입력 버튼은 기본 접힘 수동 복구 영역으로 이동
- 최소 시험 모드 옵션 추가
- 기존 v1.0.1 기능(ZIP 파싱, 이미지 저장, 중복 검사, 네이버 블로그 앱 전용 실행) 모두 유지
